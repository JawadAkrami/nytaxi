**New York Yellow Taxi – Data Cleaning (Part 1)**

I tackled the messy and massive NYC Yellow Taxi dataset from 2020, with over 24 million taxi trips! The raw data came fragmented across monthly files, with inconsistent formats, missing values, and noise everywhere. My goal was to stitch all that together into one clean and reliable dataset. I standardized data types, removed duplicate and irrelevant columns, handled missing values intelligently, and flagged obvious outliers like negative distances or nonsensical fares. This clean dataset became a solid foundation for any meaningful analysis — a crucial first step that turned a giant raw dataset into something you can actually work with.

**New York Yellow Taxi – Data Analysis (Part 2)**

Building on the cleaned dataset from Part 1, this notebook dives into exploratory analysis and modeling of NYC Yellow Taxi trips in 2020. Here, I explore key trip characteristics such as distance, fare amounts, tips, and passenger counts to uncover real patterns in how taxis were used during a very unusual year. Through summary statistics, visualizations, correlation analysis, and statistical modeling, the analysis highlights how trip distance drives fare costs, how payment behavior varies, and how demand shifts over time — even under pandemic conditions. The insights reveal meaningful relationships in the data and show how a properly cleaned dataset can unlock rich, real-world stories about urban mobility.
